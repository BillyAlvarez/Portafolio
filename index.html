Acerca de mi

Soy Licenciado en Programación Computacional y he desempeñado el rol de Analista de Sistemas durante un tiempo considerable. Durante este período, me especialicé en el análisis y desarrollo de soluciones informáticas utilizando tecnologías como Java para el backend, Spring Boot para la creación de servicios, y Angular junto con Bootstrap para el frontend. Mi experiencia incluye la participación en proyectos de desarrollo de aplicaciones web, con un enfoque en el sector de gestión financiera. Estoy comprometido con el aprendizaje continuo y estoy abierto a asumir roles que me permitan seguir creciendo profesionalmente.
